# Film.Kritik

Film.Kritik, kullanıcıların filmleri inceleyebildiği, yorum yapabildiği
ve film listeleri oluşturabildiği web tabanlı bir uygulamadır.

## Özellikler
- Kullanıcı kayıt / giriş sistemi
- Film listeleme ve detay sayfası
- Film ekleme / düzenleme (Admin)
- Yorum yapma ve yorum yönetimi
- Kategoriye göre film filtreleme

## Kullanılan Teknolojiler
- PHP
- MySQL
- HTML / CSS
- JavaScript
- Bootstrap

## Kurulum
1. Projeyi klonla
2. `database/film_kritik.sql` dosyasını phpMyAdmin ile içe aktar
3. `includes/db.php` içinden veritabanı bilgilerini düzenle
4. XAMPP / WAMP ile çalıştır

## Roller
- Kullanıcı
- Admin
